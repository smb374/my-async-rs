pub type File = IoWrapper<File>;
Expand description

File wrapper type.

This type implements:

  • create() and open(): For convenient IoWrapper<std::fs::File> creation. See std::fs::File::create() and std::fs::File::open().

For other operations, refer to IoWrapper’s documentation.

For more control on File open/creation, consider using OpenOptions to obtain a std::fs::File and wrap it using IoWrapper::from().
