UnixDatagram wrapper type.

This type implements:

  • bind(): For convenient IoWrapper<std::os::unix::net::UnixDatagram> bind address. See std::os::unix::net::UnixDatagram::bind().
  • pair(): For convenient IoWrapper<std::os::unix::net::UnixDatagram> pair creation. See std::os::unix::net::UnixDatagram::pair().
  • connect(): Async std::os::unix::net::UnixDatagram::connect().
  • recv(): Async std::os::unix::net::UnixDatagram::recv().
  • recv_from(): Async std::os::unix::net::UnixDatagram::recv_from().
  • send(): Async std::os::unix::net::UnixDatagram::send().
  • send_to(): Async std::os::unix::net::UnixDatagram::send_to().

For other operations, refer to IoWrapper’s documentation.
