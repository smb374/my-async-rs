pub type UnixListener = IoWrapper<UnixListener>;
Expand description

UnixListener wrapper type.

This type implements:

  • bind(): For convenient IoWrapper<std::os::unix::net::UnixListener> bind address. See std::os::unix::net::UnixListener::bind().
  • incoming(): Returns UnixIncoming which implements Stream for accepting loop.
  • accept(): Async std::os::unix::net::UnixListener::accept().

For other operations, refer to IoWrapper’s documentation.
