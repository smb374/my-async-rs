TcpListener wrapper type.

This type implements:

  • bind(): For convenient IoWrapper<std::net::TcpListener> bind address. See std::net::TcpListener::bind().
  • incoming(): Returns TcpIncoming which implements Stream for accepting loop.
  • accept(): Async std::net::TcpListener::accept().

For other operations, refer to IoWrapper’s documentation.
