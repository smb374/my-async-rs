UdpSocket wrapper type.

This type implements:

  • bind(): For convenient IoWrapper<std::net::UdpSocket> bind address. See std::net::UdpSocket::bind().
  • connect(): Async std::net::UdpSocket::connect().
  • peek(): Async std::net::UdpSocket::peek().
  • peek_from(): Async std::net::UdpSocket::peek_from().
  • recv(): Async std::net::UdpSocket::recv().
  • recv_from(): Async std::net::UdpSocket::recv_from().
  • send(): Async std::net::UdpSocket::send().
  • send_to(): Async std::net::UdpSocket::send_to().

For other operations, refer to IoWrapper’s documentation.
