Expand description

Token bucket based auto task yielding implementation.

An implementation using the idea from tokio’s per-task operation budget using a token bucket like algorithm.

To enable the budget use when using this runtime, use this trait on top of your module for overriding Future’s poll definition.

This trait is auto-implemented for all types that implements [FutureExt], which itself is auto-implemented for all types implementing future Future.

Note

This trait hasn’t been tested thoroughly, the trait may have little or no improvement compare to normal usage.
